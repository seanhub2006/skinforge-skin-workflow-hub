## What SkinForge Does

SkinForge is a compact game utility built around Counter-Strike 2 skin workflows. Because it is HTML-driven, it can be used as a browser-based experience or hosted locally, depending on how the repository is delivered.

This project is intended for people who want a targeted skin tool without the overhead of a large application. The emphasis is on keeping access simple, the interface approachable, and the scope narrow: handling CS2 skins through a clean, straightforward workflow.
